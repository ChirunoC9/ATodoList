# ATodoList

一个使用 Avalonia 实现的个人使用向跨平台 TodoList，支持 windows（至少为windows 10）、linux、MacOS。

前端使用 Avalonia 实现，后端使用 MongoDB 存储待办事项。

## 如何构建

```shell
$ git clone https://github.com/ChirunoC9/ATodoList.git
$ cd ATodoList
$ dotnet publish
```
